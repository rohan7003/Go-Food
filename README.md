# 🍔 GoFood - MERN Stack Food Ordering App

https://go-food-frontend-cn9dfhk35-rohans-projects-6b5be07f.vercel.app/

## 🛠 Tech Stack

- React.js
- Node.js
- Express.js

- Bootstrap

- User Signup & Login (JWT Auth)
- Location-based services
- Food ordering system
- Cart & Order management
- Responsive UI
